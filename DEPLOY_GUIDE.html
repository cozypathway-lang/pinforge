Deploy PinForge in 15 Minutes

Simple step-by-step guide — no coding experience needed

Before you start: You need 3 free accounts — GitHub (code storage), Vercel (hosting), and Anthropic (AI). All are 100% free to set up.
1

Install Node.js on your computer

Node.js is what runs the app on your computer. You only do this once.

  1. Go to nodejs.org
  2. Click the big green button that says "LTS" (recommended)
  3. Download and install it like any normal program (Next, Next, Finish)
✅ To check it worked: open your computer's terminal (or Command Prompt on Windows) and type: node --version — you should see a version number like v20.x.x
2

Create a free GitHub account

GitHub stores your code online. Vercel reads from GitHub to deploy your site.

  1. Go to github.com/signup
  2. Create a free account (takes 2 minutes)
  3. After signing in, click New → create a repository named pinforge
  4. Leave all settings as default → click Create repository
Write down: your GitHub username — you'll need it in Step 4
3

Get your Anthropic API key (the AI brain)

This is what powers the AI pin generation. Free tier gives you plenty of credits to start.

  1. Go to console.anthropic.com
  2. Sign up for a free account
  3. Click API Keys in the left menu
  4. Click Create Key → give it a name like "pinforge"
  5. Copy the key — it looks like: sk-ant-api03-...
⚠ Important: Copy the key NOW and save it somewhere (like Notepad). You can only see it once. This key stays secret — never share it or put it on social media.
4

Set up the project files

Unzip the PinForge files and set up the secret keys file.

  1. Unzip the pinforge.zip file you downloaded — this creates a pinforge folder
  2. Inside the pinforge folder, find the file named .env.example
  3. Make a copy of it and rename the copy to .env.local (remove the word "example")
  4. Open .env.local with Notepad (Windows) or TextEdit (Mac)
  5. Replace your_anthropic_api_key_here with the key you copied in Step 3
  6. Save the file
# Your .env.local should look like this: ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxx # Optional - only needed for image hosting: NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME= NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=
Note: The .env.local file is already listed in .gitignore — it will NEVER be uploaded to GitHub. Your key stays safe.
5

Test it on your computer first (optional but recommended)

Make sure everything works before deploying to the internet.

  1. Open Terminal (Mac: search "Terminal" in Spotlight) or Command Prompt (Windows: search "cmd")
  2. Type cd then drag the pinforge folder into the terminal window — this fills in the path automatically. Press Enter.
  3. Type this command and press Enter:
npm install

Wait for it to finish (1-2 minutes), then run:

npm run dev
✅ Open your browser and go to http://localhost:3000 — you should see PinForge!

Press Ctrl+C in the terminal to stop it when done testing.

6

Upload code to GitHub

Push your code to GitHub so Vercel can deploy it.

In your terminal (still in the pinforge folder), run these commands one by one:

git init git add . git commit -m "Initial PinForge setup" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/pinforge.git git push -u origin main
Replace YOUR_USERNAME with your actual GitHub username from Step 2
✅ After this, you can refresh your GitHub page and see all the files there
7

Deploy to Vercel (your live website)

Vercel turns your GitHub code into a real website. Free forever for personal projects.

  1. Go to vercel.com/signup → sign up with your GitHub account (click "Continue with GitHub")
  2. Click Add New → Project
  3. Find your pinforge repository and click Import
  4. Scroll down to Environment Variables — this is where you add your secret keys
  5. Add these variables one by one:
NameValueRequired?
ANTHROPIC_API_KEYYour key from Step 3 (sk-ant-...)Required
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAMEYour Cloudinary cloud nameOptional
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESETYour upload preset nameOptional
  1. Click Deploy
  2. Wait 1-2 minutes while Vercel builds your app
🎉 Done! Vercel gives you a live URL like pinforge.vercel.app — share it with anyone!
Auto-deploys: From now on, any time you push new code to GitHub, Vercel automatically updates your live site. No extra steps needed.
☁

Optional: Set up Cloudinary for image hosting

Without Cloudinary, images only work as previews and won't appear in the CSV URL column. With Cloudinary, every uploaded image gets a permanent shareable link.

  1. Go to cloudinary.com → create a free account Free
  2. After logging in, go to Settings → Upload → Upload presets
  3. Click Add upload preset
  4. Set Signing mode to Unsigned
  5. Click Save — copy the preset name
  6. Copy your Cloud Name from the top of the Cloudinary dashboard
  7. Add both values to Vercel's Environment Variables (or your .env.local for local testing)

Common Questions

❓ I got an error running npm install

Make sure you're in the right folder in your terminal. The folder should contain a file called package.json. If you're in the wrong folder, use cd pinforge to go into it.

❓ The AI generation isn't working after deploy

Go to your Vercel project → Settings → Environment Variables → make sure ANTHROPIC_API_KEY is there and starts with sk-ant-. After updating, click Redeploy.

❓ How do I update the app with new features?

Replace the files in your pinforge folder with the new ones, then run the same git commands from Step 6. Vercel will automatically rebuild and deploy.

❓ How much will this cost to run?

Vercel hosting: Free. Cloudinary: Free (25GB storage). Anthropic API: Pay per use — generating 100 pins costs roughly $0.05-0.10. Very affordable.
